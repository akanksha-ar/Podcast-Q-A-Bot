# Podcast Q&A Bot

AI-powered chatbot for answering questions from the Elon Musk × Nikhil Kamath podcast.

## Features

- Speech-to-text using Whisper
- Semantic search using Sentence Transformers
- Vector search using FAISS
- Returns answer with timestamp

## Technologies Used

- Python
- Whisper
- FAISS
- Sentence Transformers
